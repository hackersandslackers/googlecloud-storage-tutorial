## Getting Started

Installation is recommended via Makefile

```shell
$ git clone https://github.com/hackersandslackers/googlecloud-storage-tutorial.git
$ cd googlecloud-storage-tutorial
$ make install
```

### Usage

```shell
$ make run
```
